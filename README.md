# Визуализация графов зависимостей

Минимальный инструмент на C# для анализа и визуализации графов зависимостей npm-пакетов или тестовых локальных репозиториев.

В этом репозитории уже есть пример `sample_package.json` и утилиты для:

- построения графа зависимостей,
- вычисления порядка загрузки зависимостей,
- генерации текстового представления диаграммы в языке Mermaid и открытия его в браузере.

## Запуск

Требуется .NET 6/7/8/9 или выше.

Для запуска введите следующие команды:

```powershell
dotnet run -- <опции>
```

Или воспользуйтесь утилитой [Task](https://github.com/go-task/task) для запуска программы с различными параметрами.

Доступные сценарии Taskfile:

- `task build` - Сборка программы
- `task sample` - Построения дерева зависимости из `sample_package.json`
- `task npm` - Построения дерева зависимости из [registry.npmjs.org](https://registry.npmjs.org/)
- `task order` - Вычисление порядка загрузки зависимостей
- `task mermaid` - Визуальное представление дерева зависимостей на языке Mermaid
- `task help` - Вывод доступных опций программы

Основные опции:

- `-n, --name <name>` — имя пакета для анализа (обязательно);
- `-r, --repo <path>` — URL репозитория или путь к тестовому репозиторию (только в test-режиме);
- `-t, --test` — включить режим тестового репозитория (локальные файлы/каталог);
- `-v, --version <version>` — версия пакета (опционально);
- `-d, --max-depth <number>` — максимальная глубина обхода (0 = неограниченно);
- `-f, --filter <substring>` — фильтрация по подстроке имени пакета;
- `-o, --order` — вывести порядок загрузки зависимостей (install order);
- `-m, --mermaid` — сгенерировать диаграмму в формате Mermaid и открыть в браузере;
- `-h, --help` — показать справку.
